Html Commands

<img>
  • purpose: embeds and image into the document
  • example: <img src="https://upload.wikimedia.org/wikipedia/commons/thumb/0/0b/Cat_poster_1.jpg/390px-Cat_poster_1.jpg">
  • attributes:
    • Attribute Values Meaning
    src URL source file
    height number, percentage The intrisinic height of the image,in pixels. Must be and integer without a unit
    width number, percentage The intrinsic width of the image in pixels. Must be an integer without a unit
    alt text Defines an alternative text desctiption of the image.
<a>, </a>
  • purpose: anchor element, creates a hyperlink to web pages, files, email addresses, locations in the same page, or anything else a URL can address.
  • Example: <a href="https://en.wikipedia.org/wiki/Cat"> Wiki Cats.</a>
  • attributes:
    • Attribute Values Meaning
    href URL The URL that the hyperlink points to.
    target name_of_frame Where to display the linked URL, as the name for a browsing context (a tab, window, or <iframe>
    _blank usually a new tab, but users can configure browsers to open a new indow instead.
    _parent the parent browing context of the current one. If no parent, behaves as _self.
    _top the topmost browsing context (the "highest"context that's and ancestor of the current one). If not ancestors, behaves as_self
    _self the current browsing context. (Default)
<p>, </p>
  • purpose: new paragraph
  • attributes:
    • Attribute Values Meaning
    This element only inclues the global attributes.
    Note: The align attribue on <p> tags is obsolete and shouldn't be used.
<b>, </b>
  • purpose: to make text bold
  • Example: <b>This is bold text</b>
  • attributes:
    • Attribute Values Meaning
    This element only inclues the global attributes.
    Note: according to the HTML5 specification, the <b> tag should be used as a LAST resort when no other tag is more appropriate.
<h1> - </h6>
  • purpose: used to define HTML headings
  • Example: <h1>This is an HTML heading</h1>
  • attributes:
    • Attribute Values Meaning
    This element only inclues the global attributes.
    Note: Only use one <h1> per page - this should represent the main heading/subject for the whole page.
    Also, do not skip heading levels - start with <h1>, then use <h2>, and so on.
